- mvc代码 coding为主
  - mvc 是一种软件架构模式，将应用程序分为三个主要组件：模型（Model）、视图（View）和控制器（Controller）。
  - 模型负责处理数据和业务逻辑，视图负责用户界面的展示，控制器负责处理用户输入和协调模型和视图之间的交互。
  - mvc 模式的优势在于将应用程序的不同方面分离开来，使代码更加组织化、可维护和可扩展。
